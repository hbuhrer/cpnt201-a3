Skills

I specialize in front-end development, creating websites that look great and are easy to use. Using HTML, CSS, and JavaScript, I focus on making web experiences that engage users and work seamlessly on all devices.

I also have experience in back-end development with PHP, which allows me to build websites with advanced features. This includes things like managing databases, user accounts, and processing data to make websites more powerful and efficient.

I'm skilled in JavaScript frameworks like Vue.js and Nuxt.js, which help me create scalable web applications with dynamic interfaces and smooth transitions.

When it comes to design, I use tools like Illustrator, Photoshop, Adobe Creative Cloud, and Figma to create eye-catching graphics. Whether it's custom icons, illustrations, or layouts, I make sure they enhance the overall look and feel of the website.

My experience in front-end and back-end development, combined with my design skills, allows me to deliver websites and web applications that are not only visually appealing but also functional and user-friendly.
